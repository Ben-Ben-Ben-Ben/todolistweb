# To Do List Maker
A webpage to create simple user-created checklist tasks! This project was started to refine my pure HTML, CSS and JavaScript skills.

## Usage
- Create a new task category with the "Add Category" button. Add as many as you want! They can be rearranged by dragging the corner of the container, indicated by '<->' and a cursor change.
- To add tasks under a category, click the "Add Item" button contained within a category.
- You can customize the name of tasks and categories by clicking on their text.

## Features
- Custom, trackable tasks
- Organized group categories for each set of tasks created
- Rearrangable category windows

## Known Bugs
- Movement of categories in a multiple row configuration can lead to unexpected behaviour

## Lessons Learned
- Have a clear vision for a project **before** starting
  Scope creep was a large issue with this project. I wanted to learn too many things in one activity resulting in burnout. Honing in on a couple of key concepts will ensure I don't overwhelm myself or add unneeded complexity.
- HTML is event based
- CSS is a powerful tool for ensuring the webpage feels 'responsive'
